# Tổng quan rule, logic, luồng hoạt động, kiến trúc và chức năng các file

## 1. Kiến trúc tổng quan

- **Frontend**: Sử dụng Vue 3, cấu trúc component-based.
  - `src/components/`: Các component UI như `LoginForm`, `SocialLogin`, `RightNav`.
  - `src/views/`: Các trang chính (`Home.vue`).
  - `src/composables/`: Logic chia nhỏ (hook) cho theme, ngôn ngữ, QR code, error...
  - `src/router/`: Định tuyến, bảo vệ route.
- **Backend**: Có sử dụng Express cho route QR login (`src/router/qr-login.js`).
- **State Management**: Sử dụng composables và có thể có thêm store (ví dụ `useAuthStore`).
- **Authentication**: Đăng nhập truyền thống, đăng nhập social (Google, Facebook), QR code.

---

## 2. Rule, logic và luồng hoạt động

### 2.1. Quy tắc & logic chính

- **Theme**: Qua composable `useTheme.js` lưu theme vào storage, khởi tạo lại khi app mount (`App.vue`).
- **Ngôn ngữ**: `useLanguage.js` lưu text đa ngữ, có toggle language. Component truy cập qua hook.
- **Xác thực**: `useAuth.js` hoặc `useAuthStore.js` quản lý trạng thái đăng nhập, bảo vệ route (route guard trong `router/index.js`).
- **QR Login**: 
  - Tạo QR code động với session ID unique (`useQrCode.js`).
  - Backend Express giữ session, xác thực và tạo JWT.
  - Giao tiếp sự kiện đăng nhập thành công qua CustomEvent trên window.

### 2.2. Luồng hoạt động tiêu biểu

- User truy cập `/`: Nếu chưa đăng nhập, hiển thị `LoginForm` và `SocialLogin`.
- Đăng nhập bằng form, Google, Facebook hoặc QR.
- Nếu login thành công: route guard chuyển sang `/home`.
- Giao diện cho phép đổi theme, đổi language, đăng xuất (RightNav).
- Theme và language lưu local, khởi tạo lại khi reload.

---

## 3. Chức năng và vai trò các file

- **App.vue**: Entry point, khởi tạo theme, chứa router-view và RightNav.
- **RightNav.vue**: Đổi theme, đổi ngôn ngữ, logout.
- **LoginForm.vue**: Đăng nhập/truy cập, đăng ký, quên mật khẩu, hiển thị lỗi/thành công.
- **SocialLogin.vue**: Đăng nhập Google, Facebook, QR code.
- **useTheme.js**: Quản lý màu nền, đổi theme, lưu/khởi tạo theme.
- **useLanguage.js**: Quản lý đa ngữ, chuyển language, text cho UI.
- **useQrCode.js**: Xử lý tạo/hiển thị QR code, polling kết quả, cleanup interval, emit sự kiện thành công.
- **useError.js**: Mapping code lỗi → message user-friendly.
- **router/index.js**: Định nghĩa route, route guard dựa vào trạng thái đăng nhập.
- **router/qr-login.js**: API backend đăng nhập QR, quản lý session, xác thực, phát JWT.
- **firebase.js**: Khởi tạo firebase, provider Google/Facebook.
